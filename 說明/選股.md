# XS 內建函數參考

### 融券最後回補日

```xs
Value1 = GetField("ShortSaleFinalRecallDate");
```

**說明:**

事件\n數值\n選股\n函數\n單一\n台股\n最近一次融券最後回補日的日期。\n欄位格式為西元年月日，例如20221101。\n系統依照最近一次公布的股東會，除權息，現增，或是減資，回傳最接近的融券最後回補日。

### 除權息值

```xs
Value1 = GetField("ExDividendRightValue");
```

**說明:**

事件\n元\n數值\n選股\n函數\n單一\n台股\n最近一次公佈的除權或是除息的數值。\n如果股票在同一日除權+除息的話，則回傳配發的現金與配發的股票的加總，否則則回傳最近一次的除權值，或是除息值。\n僅支援台股。\n例如：如果股票同一日除權1元(每1000股配發100股)且除息0.5元的話，則數值為1.5。\n可以檢查GetFieldDate("除權息值")是否等於0來判斷是否有資料。

### 除權值

```xs
Value1 = GetField("ExRightValue");
```

**說明:**

事件\n元\n數值\n選股\n函數\n單一\n台股\n美(股票)\n最近一次公佈的除權值。\n台股的話單位是元(每1股配發幾股股票)，美股的話則是除權(Split)的比例。\n舉例而言，在台股，如果每股配發0.5股的話(每1000股配發500股)，那數值是0.5。\n在美股，如果分割的比例是1:2的話(每1股變成0.5股)，則數值為0.5，如果是2:1的話(每1股變成2股)，則數值為2。\n請注意：這個數值可能是即將除權的數值(例如已經公佈要除權，可是尚未除權)，也可能是過去一次除權的數值(例如已經除權，可是下次除權日尚未確定)。\n可以檢查GetFieldDate("除權值")是否等於0來判斷是否有資料。
