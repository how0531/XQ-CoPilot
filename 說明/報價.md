# 報價欄位 (Quote)

> 本文件列出所有 GetQuote 支援的欄位及其說明。

### 總委賣

**函數網址**: [總委賣](https://xshelp.xq.com.tw/XSHelp/?HelpName=%E7%B8%BD%E5%A7%94%E8%B3%A3&group=QFIVE)

* **語法**:
    ```xs
    Value1 = GetQuote("總委賣");
    Value1 = GetQuote("SumAskSize");
    Value1 = q_SumAskSize;
    ```
* **分類**: 五檔統計
* **支援腳本**: 警示, 交易, 函數
* **單位**: 台股:張；大盤/類股:張；期貨/選擇權:口；其他市場股票:股
* **格式**: 數值
* **說明**: 委託簿上的前五檔委賣數量的加總。
數值單位依照商品而異。
如果是大盤或是類股指數的話，數值是成分股所有商品的總委賣張數的加總。

### 委賣

**函數網址**: [委賣](https://xshelp.xq.com.tw/XSHelp/?HelpName=%E5%A7%94%E8%B3%A3&group=QFIVE)

* **語法**:
    ```xs
    Value1 = GetQuote("委賣");
    Value1 = GetQuote("BestAskSize");
    Value1 = GetQuote("AskSize");
    Value1 = GetQuote("最佳委賣");
    Value1 = q_BestAskSize;
    Value1 = q_AskSize;
    ```
* **分類**: 五檔統計
* **支援腳本**: 警示, 交易, 函數
* **單位**: 台股:張；期貨/選擇權:口；其他市場股票:股
* **格式**: 數值
* **說明**: 目前委託簿內賣方最低賣價的委託量。
數值單位依照商品而異。
如果是台期貨商品的話，則比對五檔委託簿上面最低賣價以及衍生委託單的委賣價：

如果五檔的最低賣價比較低，則是取此賣價的委託量，
如果衍生委託單的委賣價比較低，則是取衍生委託單的委賣量，
如果兩者相同，則是取五檔最低委賣價的委託量以及衍生委託單的委賣量的加總。

### 參考價

    ```xs

    Value1 = q_RefPrice;
    ```
* **分類**: 常用
* **支援腳本**: 警示, 交易, 函數
* **單位**: 元
* **格式**: 數值
* **說明**: 當日的參考價。也可以使用 GetField("RefPrice", "D") 來表示。
一般而言是前一交易日的收盤價，如果股票商品有除權息的話，則是除權息後的參考價。
除權息的計算公式如下(實際數值以交易所公佈為主)：
1.除息參考價為股價－股息
2.除權參考價為股價／(1+配股率)
3.同時除權息參考價為 (股價-股息)／(1+配股率)

### 估計量

**函數網址**: [估計量](https://xshelp.xq.com.tw/XSHelp/?HelpName=%E4%BC%B0%E8%A8%88%E9%87%8F&group=QOFTEN)

* **語法**:
    ```xs
    Value1 = GetQuote("估計量");
    Value1 = GetQuote("EstimatedTotalVolume");
    Value1 = q_EstimatedTotalVolume;
    ```
* **分類**: 常用
* **支援腳本**: 警示, 交易, 函數
* **單位**: 個股:張，大盤:元
* **格式**: 數值
* **說明**: 當日收盤的預估成交量。也可以使用 GetField("估計量", "D") 來表示。
大盤或是個股的估計的方式是統計過去一段時間內每日盤中的分鐘累計成交量，算出每分鐘成交量的分佈比例，然後依照當日開盤迄今的累計成交量，來推算收盤時可能的成交量。
類股商品（細產業指標、主題指標）的計算方式為成分股預估量乘以均價的加總。
